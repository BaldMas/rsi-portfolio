# Solution: сканер ротации портфеля

## Задача
Оценивать ротацию между монетами портфеля по нескольким метрикам.

## Решение
Два скрипта:
- `ratio_rsi.py` - детальный анализ пары (RSI daily/weekly + Z-score + Relative Performance)
- `portfolio_rotation.py` - батч-скан всего портфеля, итоговый рейтинг сигналов

## Метрики
1. **RSI соотношения** (daily 14, weekly 14) - перекупленность/перепроданность относительного курса
2. **Z-score 90д** - статистическая экстремальность ratio: >+1.5 = аномально дорого, <-1.5 = аномально дёшево
3. **Relative Strength 7d/30d** - разница % изменения монеты vs цели за период

## Итоговый score
`score = (RSI_D-50)/10 + (RSI_W-50)/10*1.5 + Z90*2 + (-RS30/10)`
- > +4: ротировать
- +2 до +4: смотреть
- < -3: ждать (продаёшь на историческом дне)

## Данные
Binance klines (бесплатно, без ключа), fallback Gate.io для XCH.
